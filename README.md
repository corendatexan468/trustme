## 🖥️ What trustme does

trustme is a Windows tool that helps trigger TrustedInstaller-level actions through the DISM API and thread impersonation. In simple terms, it is made for system tasks that need higher access than a standard user account can provide.

This tool is built for users who need to work with protected Windows components and system-level tasks in a controlled way. It fits into setups where TrustedInstaller access is part of the workflow.
